# Screenshots

Use this folder for project screenshots used in README, report, and PPT.

Recommended filenames:

| File | Purpose |
|---|---|
| `dashboard-ready.png` | Dashboard before camera starts |
| `live-detection.png` | Webcam face detection with bounding box |
| `history-section.png` | Detection history table |
| `backend-health.png` | Backend health API response |

Current generated screenshot:

```text
screenshots/dashboard-ready.png
```

For webcam detection screenshots, start the app, click Start Camera, and save a
browser screenshot manually because headless browser tools cannot use your
physical webcam reliably.
