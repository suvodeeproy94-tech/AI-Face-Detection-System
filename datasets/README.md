# Dataset Folder

This folder is prepared for face detection datasets.

For the first working version, the project does not need WIDER FACE or FDDB
immediately because OpenCV already provides a pretrained Haar Cascade model.

Use this folder later when you want to test accuracy more formally.

## Expected Local Dataset Paths

```text
D:\AI-Face-Detection-System\datasets\WIDER_FACE
D:\AI-Face-Detection-System\datasets\FDDB
```

## Important Git Rule

Do not commit full dataset images to GitHub. These files are very large and
should stay only on the local machine.
